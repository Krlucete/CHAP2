Help with Gmaj

This is a special version of Gmaj that has been extended with features tailored for examining results from the conversion detection pipeline in the CHAP 2 software suite.

  • For a tutorial on using these new features, see Viewing Conversion Evidence with Gmaj.

  • For an example of using Gmaj to view calls from the orthology mapping pipeline, see Examining the Orthology Results.

  • For basic information about using Gmaj, please see the documentation for its current main release (which is rather old but still mostly applicable), especially the page "Starting and Running Gmaj".
